### JAP 알게된것들
  * @Entity: 영속성 컨텍스트에 의해 관리되는 클래스를 의미, 단순하게 테이블에 매핑되는 객체라고 생각
  * @Table: 엔티티에 매핑되는 데이터베이스 테이블 명
  * @Id: 기본키가 될 변수를 의미
